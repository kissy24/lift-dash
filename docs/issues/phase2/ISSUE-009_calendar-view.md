# [ISSUE-009] カレンダービュー

## 概要

記録一覧に月間カレンダーを追加し、トレーニング実施日をハイライトして日付別詳細へ移動できるようにする。

## 背景・目的

トレーニング頻度と継続状況を時系列で把握し、対象日の記録へ素早くアクセスできるようにする。

## 受け入れ条件 (Acceptance Criteria)

- [ ] `/log`で月間カレンダーを表示できる
- [ ] トレーニング記録のある日を視覚的に区別できる
- [ ] 記録のある日を選ぶと`/log/[date]`へ移動できる
- [ ] 前月・翌月へ移動できる
- [ ] 日付別一覧表示も引き続き利用できる
- [ ] モバイルとPCで操作できる
- [ ] 対応するテストがすべてグリーン

## 技術詳細

- date-fnsで月初・月末・週単位の表示日を計算する
- URL query（例: `?month=2026-07`）を表示月の状態源とする
- 不正な月queryは当月へフォールバックする
- 新規カレンダー依存パッケージは追加せず、既存UIとdate-fnsで実装する
