# [ISSUE-007] プリセットからの一括入力

## 概要

記録入力画面でプリセットを選択し、登録済みの種目とデフォルトセットを一括展開できるようにする。

## 背景・目的

毎回同じ種目とセット数を手動追加する操作を減らし、日常的な記録入力を短時間で完了できるようにする。

## 受け入れ条件 (Acceptance Criteria)

- [ ] `/log/new`でプリセットを選択できる
- [ ] 選択したプリセットの種目、重量、レップ数、セット数がフォームへ展開される
- [ ] 展開後も種目・セットの追加、削除、値の編集ができる
- [ ] 別のプリセットを選択する際に未保存入力の置換を確認できる
- [ ] プリセットを使わない手動入力を引き続き利用できる
- [ ] 対応するテストがすべてグリーン

## 技術詳細

- ページのServer Componentでプリセットとアイテムを取得する
- Client Componentには公開可能な初期値だけを渡す
- 保存処理はISSUE-004のServer ActionとDB関数を再利用する
- プリセット選択後の値は通常フォーム値として扱い、プリセット変更が既存記録へ影響しないようにする
